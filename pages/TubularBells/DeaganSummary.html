Deagan Tower Chimes Summary

This is a state-by-state summary of the current status of tower tube instruments produced by the J.C.Deagan Company, according to the place of original installation.  It is based in part on the related page which lists all original Deagan Tower Tube Installations; that page provides the figures listed in the column headed "Original installations."  The figures for "surviving installations" are based on current status to the extent that it is known, including relocations.  So it is possible to have survivals exceed originals.  When a surviving number is shown as a range of values, it indicates that the status of some of the original installations for that state or province is definitely unknown.
